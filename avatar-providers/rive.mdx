## Rive

To integrate Rive avatars, you can explore all the great creative work on [rive community](https://rive.app/community/).

Here is an exmaple of using a customized avatar from [@edofaravelli](https://rive.app/community/5160-10364-hey-ho-lets-go/)

```jsx
import { ElevenLabVoiceService } from '@avatechai/avatars';
import { AvatarDisplay, AvatarChatBox } from '@avatechai/avatars/react';

const riveAvatarUrl =
  'https://avatech-avatar-dev1.nyc3.cdn.digitaloceanspaces.com/Marty.riv';
export default function App() {
  return (
    <AvatarDisplay src={riveAvatarUrl}>
        <AvatarChatBox />
    </AvatarDisplay>
  )
}
```
