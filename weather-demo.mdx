## Requirements

Environment variables are required to run this project. You can create a `.env` file in the root directory and add the following variables.

```bash
# https://tryleap.ai/ get your API key here
LEAP_API_KEY=
OPENAI_API_KEY=
# https://openweathermap.org/ get your API key here
OPENWEATHER_API_KEY=
NEXT_PUBLIC_ELEVEN_LABS_API_KEY=
```

## Features

- Function calling with Open AI
- Global Weather API - [Open Weather](https://openweathermap.org)
- Leap AI - Image Generator - [Leap AI](https://tryleap.ai)
