## Integrate with Vercel AI SDK

<Note title='Note' type='note'>
  This demo shows how to integrate with Vercel AI SDK
  https://sdk.vercel.ai/docs/guides/frameworks/nextjs-app to create a LLM backend.
</Note>

Via Vercel AI SDK, you can leverage the power of any LLM backend that the `ai` sdk support. Which includes, OpenAI, LangChain, and more.

For a complete demo and GitHub repo, you can check this out directly. 
- [Live Demo](https://avatars-nextjs-demo.vercel.app/)
- [Interactive Demo](https://avatars-nextjs-demo.vercel.app/demo)
